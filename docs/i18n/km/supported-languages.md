## បន្ថែមភាសា

ដើម្បីបន្ថែមការគាំទ្រសម្រាប់ភាសាថ្មី:

1. បន្ថែមកូដភាសា និងឈ្មោះដែលបង្ហាញទៅក្នុងប្រព័ន្ធឧបករណ៍ភាសា។
2. Add or map a font in `src/co_op_translator/fonts/font_language_mappings.yml`.
3. សាកល្បងលទ្ធផលបកប្រែ Markdown និងរូបភាព។
4. បើក pull request ជាមួយនឹងការរកមាប៉ារ៉ាម៉ែត្រ និងកំណត់សម្គាល់ការផ្ទៀងផ្ទាត់។
