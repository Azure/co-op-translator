# ಬೆಂಬಲಿತ ಭಾಷೆಗಳು

Co-op Translator ಕೆಳಗಿನ ಭಾಷಾ ಕೋಡ್ಗಳನ್ನು ಪಠ್ಯ, ನೋಟ್‌ಬುಕ್ ಮತ್ತು ಚಿತ್ರ ಭಾಷಾಂತರ ಔಟ್‌ಪುಟ್‌ಗಳಿಗೆ ಬೆಂಬಲಿಸುತ್ತದೆ.

ನೀವು ಹೊಸ ಭಾಷೆಯನ್ನು ಸೇರಿಸಬೇಕಾದರೆ, `src/co_op_translator/fonts/` ಒಳಗೆ ಭಾಷೆ ಮತ್ತು ಫಾಂಟ್ ಮ್ಯಾಪ್‌ಗಳನ್ನು ನವೀಕರಿಸಿ ಮತ್ತು ಪುಲ್ ರಿಕ್ವೆಸ್ಟ್ ತೆರೆಯುವುದಕ್ಕೆ ಮೊದಲು ಭಾಷೆಯನ್ನು ಪರೀಕ್ಷಿಸಿ.

## ಭಾಷೆ ಸೇರಿಸಿ

To add support for a new language:

1. ಭಾಷಾ ಕೋಡ್ ಮತ್ತು ಪ್ರದರ್ಶನ ಹೆಸರುಗಳನ್ನು language utilities ಗೆ ಸೇರಿಸಿ.
2. `src/co_op_translator/fonts/font_language_mappings.yml` ನಲ್ಲಿ ಫಾಂಟ್ ಅನ್ನು ಸೇರಿಸಿ ಅಥವಾ ಮ್ಯಾಪ್ ಮಾಡಿ.
3. Markdown ಮತ್ತು ಚಿತ್ರದ ಭಾಷಾಂತರ ಔಟ್‌ಪುಟ್ ಅನ್ನು ಪರೀಕ್ಷಿಸಿ.
4. ಮ್ಯಾಪಿಂಗ್ ಮತ್ತು ಮಾನ್ಯತಾ ಟಿಪ್ಪಣಿಗಳೊಂದಿಗೆ ಪುಲ್ ರಿಕ್ವೆಸ್ಟ್ ತೆರೆಯಿರಿ.
