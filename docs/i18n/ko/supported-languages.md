## 언어 추가

새 언어를 지원하려면:

1. 언어 유틸리티에 언어 코드와 표시 이름을 추가합니다.
2. `src/co_op_translator/fonts/font_language_mappings.yml`에 폰트를 추가하거나 매핑합니다.
3. Markdown 및 이미지 번역 출력을 테스트합니다.
4. 매핑 및 검증 메모와 함께 풀 리퀘스트를 엽니다.
