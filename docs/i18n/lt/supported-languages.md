## Pridėti kalbą

Norėdami pridėti naują kalbą:

1. Pridėkite kalbos kodą ir rodomą pavadinimą į kalbų įrankius.
2. Pridėkite arba susiekite šriftą faile `src/co_op_translator/fonts/font_language_mappings.yml`.
3. Išbandykite Markdown ir vaizdų vertimo išvestis.
4. Atidarykite pull request su žemėlapiu ir patikrinimo pastabomis.
