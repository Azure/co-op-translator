# Azure AI ക്രമീകരണം

ഈ മാർഗനിർദ്ദേശം ഉപയോഗിക്കുക όταν നിങ്ങൾ ടെക്സ്റ്റ് വിവർത്തനത്തിനായി Azure OpenAIയും ഇമേജ് ടെക്സ്റ്റ് എക്സ്ട്രാക്ഷനത്തിനായി Azure AI Visionഉം ക്രമീകരിക്കേണ്ടതുണ്ടെങ്കിൽ.

## ആവശ്യമുള്ളതെല്ലാം

- ഒരു Azure സബ്സ്ക്രിപ്ഷൻ.
- Azure AI വിഭവങ്ങളും മോഡൽ ഡിപ്പ്ലോയ്മെന്റുകളും സൃഷ്ടിക്കാനോ ഉപയോഗിക്കാനോ അനുവദനീയമായ അനുമതി.
- Azure AI ഫൗണ്ട്രിയിൽ (Azure AI Foundry) ഒരു പ്രോജക്ട് അല്ലെങ്കിൽ Azure OpenAIക്കും Azure AI Visionക്കും സമാനമായ ആക്സസ്.

## Azure AI പ്രോജക്ട് സൃഷ്ടിക്കുക

1. തുറക്കുക [Azure AI ഫൗണ്ട്രി](https://ai.azure.com).
2. ഒരു പ്രോജക്ട് സൃഷ്ടിക്കുകയോ തിരഞ്ഞെടുക്കുകയോ ചെയ്യുക.
3. പ്രോജക്ടിനായി ഒരു AI ഹബ് സൃഷ്ടിക്കുകയോ തിരഞ്ഞെടുക്കുകയോ ചെയ്യുക.
4. സൃഷ്ടിച്ചതിനു ശേഷം പ്രോജക്ട് അവലോകനം തുറക്കുക.

## Azure OpenAI മോഡൽ ഡിപ്ലോയ് ചെയ്യുക

1. പ്രോജക്ടിൽ **മോഡലുകൾ + എൻഡ്‌പോയിന്റുകൾ** തുറക്കുക.
2. **മോഡൽ ഡിപ്ലോയ് ചെയ്യുക** തിരഞ്ഞെടുക്കുക.
3. `gpt-4o` പോലുള്ള ഒരു GPT മോഡൽ തിരഞ്ഞെടുക്കുക.
4. മോഡൽ ഡിപ്ലോയ് ചെയ്യുക.
5. എൻഡ്‌പോയിന്റ്, ഡിപ്ലോയ്മെന്റ് പേര്, മോഡൽ പേര്, API കീ, API വേർഷൻ എന്നിവ രേഖപ്പെടുത്തുക.

!!! note
    Azure OpenAI API വേർഷൻ Azure AI Foundry-ൽ കാണിക്കുന്ന മോഡൽ വേർഷനിൽനിന്നും വേറാണ്. നിങ്ങളുടെ ഡിപ്ലോയ്മെന്റിനായി പിന്തുണയുള്ള ഒരു API വേർഷൻ തിരഞ്ഞെടുക്കുക.

## Azure AI Vision ക്രമീകരിക്കുക

ചിത്ര വിവർത്തനം നടത്തുന്നതിന് മുൻപ് ഉറവിട ചിത്രങ്ങളിൽ നിന്നുള്ള ടെക്സ്റ്റ് എക്‌സ്‌ട്രാക്ഷനിന് Azure AI Vision ഉപയോഗിക്കുന്നു, ശേഷം ആ ടെക്സ്റ്റ് വിവർത്തനം ചെയ്യപ്പെടും.

നിങ്ങളുടെ Azure AI പ്രോജക്ടിൽ Azure AI Services കീയും എൻഡ്‌പോയിന്റും കണ്ടെത്തുക.

![Azure AI സേവന വിവരങ്ങൾ കണ്ടെത്തുക](../../assets/find-azure-ai-info.png)

രേഖപ്പെടുത്തുക:

- Azure AI സർവീസ് എൻഡ്‌പോയിന്റ്
- Azure AI സർവീസ് API കീ

## പരിസ്ഥിതി വേരിയബിൾസ്

നിങ്ങളുടെ ക്രെഡൻഷ്യലുകൾ നിങ്ങളുടെ `.env` ഫയലിലോ CI രഹസ്യങ്ങളിലോ ചേർക്കുക.

```bash
# ചിത്ര വിവർത്തനത്തിനായി ആവശ്യമായ Azure AI Vision
AZURE_AI_SERVICE_API_KEY="..."
AZURE_AI_SERVICE_ENDPOINT="https://<resource>.cognitiveservices.azure.com/"

# ടെക്സ്റ്റ് വിവർത്തനത്തിനായി ആവശ്യമായ Azure OpenAI
AZURE_OPENAI_API_KEY="..."
AZURE_OPENAI_ENDPOINT="https://<resource>.openai.azure.com/"
AZURE_OPENAI_MODEL_NAME="gpt-4o"
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME="<deployment>"
AZURE_OPENAI_API_VERSION="2024-12-01-preview"
```

Co-op Translator ഐച്ഛികമായി ഫോള്ബാക്ക് ക്രെഡൻഷ്യൽ സെറ്റുകൾക്കും പിന്തുണ നൽകുന്നു. `_1` അല്ലെങ്കിൽ `_2` പോലുള്ള സഫിക്സുകളുമായി ഒരു പൂർണ്ണprovidername സെറ്റ് മടക്കിച്ച് സൃഷ്ടിക്കുക; ഒരു ഫോള്ബാക്ക് സെറ്റിലുള്ള എല്ലാ വേരിയബിളുകളും ഒരേ സഫിക്സ് പങ്കിടണം.

```bash
AZURE_OPENAI_API_KEY_1="..."
AZURE_OPENAI_ENDPOINT_1="https://<resource-1>.openai.azure.com/"
AZURE_OPENAI_MODEL_NAME_1="gpt-4o"
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME_1="<deployment-1>"
AZURE_OPENAI_API_VERSION_1="2024-12-01-preview"
```

## അടുത്ത ചുവടുകൾ

- നിങ്ങളുടെ ലോക്കൽ അല്ലെങ്കിൽ CI പരിസ്ഥിതി വേരിയബിൾസ് ക്രമീകരിക്കാൻ [Configuration](configuration.md) ന് തിരികെ പോകുക.
- വിവർത്തന കമാൻഡുകൾക്കായി [CLI റഫറൻസ്](cli.md) ഉപയോഗിക്കുക.
- വിവർത്തന പുൾ അഭ്യർത്ഥനകൾ ഓട്ടോമേറ്റ് ചെയ്യാൻ [GitHub Actions](github-actions.md) ഉപയോഗിക്കുക.
