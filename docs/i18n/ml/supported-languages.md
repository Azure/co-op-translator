## ഒരു ഭാഷ ചേർക്കുക

ഒരു പുതിയ ഭാഷക്ക് പിന്തുണ ചേർക്കാൻ:

1. ഭാഷാ കോഡ് והപ്രദർശന പേര് ഭാഷാ ഉപകരണങ്ങളിലേക്ക് ചേർക്കുക.
2. `src/co_op_translator/fonts/font_language_mappings.yml` ൽ ഫോണ്ട് ചേർക്കുക അല്ലെങ്കിൽ മാപ്പ് ചെയ്യുക.
3. Markdown மற்றும் ഇമേജ് തർജ്ജമ ഔട്ട്പുട്ട് പരിശോധന നടത്തുക.
4. മാപ്പിംഗ് 및 സാധുതാ കുറിപ്പുകൾ സഹിതം ഒരു പുൽ റിക്വസ്റ്റ് തുറക്കുക.
