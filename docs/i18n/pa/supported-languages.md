## ਭਾਸ਼ਾ ਜੋੜੋ

To add support for a new language:

1. ਭਾਸ਼ਾ ਕੋਡ ਅਤੇ ਡਿਸਪਲੇ ਨਾਮ language utilities ਵਿੱਚ ਜੋੜੋ।
2. ਇੱਕ ਫੋਂਟ ਸ਼ਾਮਲ ਕਰੋ ਜਾਂ ਮੈਪ ਕਰੋ `src/co_op_translator/fonts/font_language_mappings.yml`.
3. Markdown ਅਤੇ ਚਿੱਤਰ ਅਨੁਵਾਦ ਆਉਟਪੁੱਟ ਟੈਸਟ ਕਰੋ।
4. ਮੈਪਿੰਗ ਅਤੇ ਵੈਲਿਡੇਸ਼ਨ ਨੋਟਸ ਦੇ ਨਾਲ ਇੱਕ ਪਲ ਰਿਕਵੇਸਟ ਖੋਲ੍ਹੋ।
