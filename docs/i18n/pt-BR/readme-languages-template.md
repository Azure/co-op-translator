# Modelo de README de Idiomas

O Co-op Translator pode manter uma tabela de idiomas no README para repositórios que publicam conteúdo traduzido.

Use os marcadores abaixo quando quiser que o Co-op Translator substitua toda a seção a cada execução de tradução. Remova os marcadores se preferir manter manualmente um subconjunto personalizado.

````markdown
### Multi-Language Support

#### Supported by [Co-op Translator](https://github.com/Azure/co-op-translator)

> **Prefer to Clone Locally?**
>
> This repository includes many language translations, which can significantly increase download size. To clone without translations, use sparse checkout:
>
> ```bash
> git clone --filter=blob:none --sparse https://github.com/org/repo.git
> cd repo
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```

<!-- CO-OP TRANSLATOR LANGUAGES TABLE END -->
````

Personalize a URL do repositório sparse-checkout durante a tradução:

```bash
translate -l "ko" --repo-url "https://github.com/org/repo.git"
```

Se os marcadores estiverem presentes, o Co-op Translator pode atualizar a tabela gerada quando idiomas forem adicionados ou alterados.
