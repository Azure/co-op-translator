## ஒரு மொழியைச் சேர்க்க

To add support for a new language:

1. மொழி குறியீடு மற்றும் காட்சி பெயரை language utilities-இல் சேர்க்கவும்.
2. `src/co_op_translator/fonts/font_language_mappings.yml` இல் ஒரு எழுத்துருவைச் சேர்க்க அல்லது வரைபடுத்தவும்.
3. Markdown மற்றும் பட மொழிபெயர்ப்பு வெளியீட்டை சோதிக்கவும்.
4. மாப்பிங் மற்றும் சரிபார்ப்பு குறிப்புகளுடன் ஒரு pull request திறக்கவும்.
