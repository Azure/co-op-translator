## زبان شامل کریں

To add support for a new language:

1. زبان کوڈ اور ڈسپلے نام زبان یوٹیلیٹیز میں شامل کریں۔
2. `src/co_op_translator/fonts/font_language_mappings.yml` میں ایک فونٹ شامل کریں یا نقشہ بنائیں۔
3. مارک ڈاؤن اور تصویر کے ترجمے کے نتائج کی جانچ کریں۔
4. میپنگ اور تصدیقی نوٹس کے ساتھ ایک پل ریکویسٹ کھولیں۔
