# সমর্থিত ভাষাসমূহ

নিচের টেবিলটি বর্তমানে **Co-op Translator** দ্বারা সমর্থিত ভাষাসমূহের তালিকা দেয়। এতে ভাষার কোড, ভাষার নাম এবং প্রতিটি ভাষার সাথে সম্পর্কিত যেকোনো পরিচিত সমস্যা অন্তর্ভুক্ত রয়েছে। আপনি যদি নতুন কোনো ভাষার সমর্থন যোগ করতে চান, তাহলে অনুগ্রহ করে সংশ্লিষ্ট ভাষার কোড, নাম এবং উপযুক্ত ফন্ট `src/co_op_translator/fonts/` এর মধ্যে অবস্থিত `font_language_mappings.yml` ফাইলে যোগ করুন এবং পরীক্ষা করার পর একটি পুল রিকোয়েস্ট জমা দিন।
