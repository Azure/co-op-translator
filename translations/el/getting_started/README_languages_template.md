# 🌐 Υποστήριξη Πολλών Γλωσσών (Πρότυπο)

Υπεύθυνοι: Το μπλοκ παρακάτω είναι ένα παράδειγμα για "όλες τις γλώσσες" που διαχειρίζεται ο Co‑op Translator.

- Αν θέλετε ο Co‑op Translator να κρατά αυτή τη λίστα πλήρως ενημερωμένη αυτόματα όταν εκτελείτε `translate -l "all"`, αφήστε τα δύο σχόλια όπως είναι.
- Αν θέλετε να εμφανίσετε μόνο ένα υποσύνολο γλωσσών, διαγράψτε τα δύο σχόλια και αφαιρέστε τις γλώσσες που δεν θέλετε να εμφανίζονται. Μετά την αφαίρεση των σχολίων, ο Co‑op Translator δεν θα αντικαθιστά πλέον αυτό το τμήμα αυτόματα.
