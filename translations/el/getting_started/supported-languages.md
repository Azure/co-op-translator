# Υποστηριζόμενες γλώσσες

Ο παρακάτω πίνακας απαριθμεί τις γλώσσες που υποστηρίζονται αυτή τη στιγμή από το **Co-op Translator**. Περιλαμβάνει τους κωδικούς γλωσσών, τα ονόματα των γλωσσών και τυχόν γνωστά προβλήματα που σχετίζονται με κάθε γλώσσα. Αν θέλετε να προσθέσετε υποστήριξη για μια νέα γλώσσα, παρακαλούμε προσθέστε τον αντίστοιχο κωδικό γλώσσας, το όνομα και την κατάλληλη γραμματοσειρά στο αρχείο `font_language_mappings.yml` που βρίσκεται στο `src/co_op_translator/fonts/` και υποβάλετε ένα pull request μετά από δοκιμές.
