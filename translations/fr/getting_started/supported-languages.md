## Ajouter une nouvelle langue

Pour ajouter la prise en charge d'une nouvelle langue :

1. Rendez-vous sur [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml).
2. Ajoutez le code langue, le nom et le nom du fichier de police approprié. Assurez-vous d’inclure l’attribut `rtl` si la langue s’écrit de droite à gauche.
3. Si vous devez utiliser une nouvelle police, vérifiez que celle-ci est libre d’utilisation dans des projets open source en consultant ses conditions de licence et droits d’auteur. Après vérification, ajoutez le fichier de police dans le répertoire `src/co_op_translator/fonts/`.
4. Testez vos modifications localement pour vous assurer que la nouvelle langue est bien prise en charge.
5. Soumettez une Pull Request avec vos modifications en indiquant l’ajout de la nouvelle langue dans la description de la PR.

Exemple :

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
