# 🌐 תמיכה בריבוי שפות (תבנית)

מתחזקים: הקטע למטה הוא דוגמה ל"כל השפות" שמנוהלת על ידי Co‑op Translator.

- אם ברצונכם ש-Co‑op Translator יעדכן את הרשימה הזו אוטומטית לחלוטין כאשר תריצו `translate -l "all"`, השאירו את שני סימני ההערה כפי שהם.
- אם ברצונכם להציג רק חלק מהשפות, מחקו את שני סימני ההערה והסירו כל שפה שאינכם רוצים להציג. לאחר הסרת הסימנים, Co‑op Translator לא יחליף עוד אוטומטית את הקטע הזה.

**הצהרת אחריות**:
מסמך זה תורגם באמצעות שירות תרגום מבוסס בינה מלאכותית [Co-op Translator](https://github.com/Azure/co-op-translator). למרות שאנו שואפים לדיוק, יש לקחת בחשבון כי תרגומים אוטומטיים עשויים להכיל טעויות או אי-דיוקים. יש לראות במסמך המקורי בשפתו המקורית כמקור הסמכותי. למידע קריטי, מומלץ לפנות לתרגום מקצועי על ידי אדם. איננו אחראים לכל אי-הבנות או פירושים שגויים הנובעים מהשימוש בתרגום זה.
