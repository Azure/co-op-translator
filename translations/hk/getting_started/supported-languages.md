## 新增語言指引

如果想新增支援嘅語言：

1. 前往 [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml)。
2. 加入語言代碼、名稱同適用嘅字型檔名。如果語言係由右至左，記得加 `rtl` 屬性。
3. 如果要用新字型，請先確定該字型可免費用於開源項目，檢查授權同版權條款。確認後，將字型檔加入 `src/co_op_translator/fonts/` 目錄。
4. 喺本地測試改動，確保新語言支援正常。
5. 提交拉取請求，並喺 PR 描述中標明新增語言。

範例：

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
