# Podržani jezici

Tablica u nastavku prikazuje jezike koje trenutno podržava **Co-op Translator**. Uključuje kodove jezika, nazive jezika i poznate probleme povezane s pojedinim jezicima. Ako želite dodati podršku za novi jezik, molimo dodajte odgovarajući kod jezika, naziv i odgovarajući font u datoteku `font_language_mappings.yml` koja se nalazi u `src/co_op_translator/fonts/` te po testiranju pošaljite pull request.
