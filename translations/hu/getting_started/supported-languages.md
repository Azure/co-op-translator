# Támogatott nyelvek

Az alábbi táblázat a **Co-op Translator** által jelenleg támogatott nyelveket tartalmazza. Feltünteti a nyelvkódokat, a nyelvek neveit, valamint az adott nyelvhez kapcsolódó ismert problémákat. Ha új nyelvet szeretnél hozzáadni, kérjük, vidd fel a megfelelő nyelvkódot, nevet és a megfelelő betűtípust a `font_language_mappings.yml` fájlba, amely a `src/co_op_translator/fonts/` mappában található, majd tesztelés után küldj be egy pull requestet.
