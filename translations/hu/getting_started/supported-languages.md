## Új nyelv hozzáadása

Új nyelv támogatásának hozzáadásához:

1. Lépj a [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml) fájlhoz.
2. Add hozzá a nyelvkódot, nevet és a megfelelő betűtípus fájlnevet. Ha a nyelv jobbról balra íródik, mindenképp add meg az `rtl` attribútumot.
3. Ha új betűtípust szeretnél használni, győződj meg róla, hogy az szabadon felhasználható nyílt forráskódú projektekben, és ellenőrizd a licenc- és szerzői jogi feltételeket. Ezek után add hozzá a betűtípus fájlt a `src/co_op_translator/fonts/` könyvtárhoz.
4. Teszteld a változtatásokat helyben, hogy meggyőződj arról, hogy az új nyelv megfelelően támogatott.
5. Küldj egy Pull Requestet a módosításokkal, és jelezd az új nyelv hozzáadását a PR leírásában.

Példa:

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
