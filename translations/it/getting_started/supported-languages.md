## Aggiungere una nuova lingua

Per aggiungere il supporto a una nuova lingua:

1. Vai su [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml).
2. Aggiungi il codice lingua, il nome e il file del font appropriato. Assicurati di includere l'attributo `rtl` se la lingua si scrive da destra a sinistra.
3. Se devi usare un nuovo font, verifica che sia gratuito per progetti open-source controllandone la licenza e i termini di copyright. Dopo la verifica, aggiungi il file del font nella directory `src/co_op_translator/fonts/`.
4. Testa le modifiche localmente per assicurarti che la nuova lingua sia supportata correttamente.
5. Invia una Pull Request con le tue modifiche e indica l'aggiunta della nuova lingua nella descrizione della PR.

Esempio:

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
