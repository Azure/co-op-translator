# 🌐 多言語対応（テンプレート）

メンテナーの方へ：以下のブロックは、Co‑op Translatorによって管理されている「全言語」例です。

- `translate -l "all"`を実行して、Co‑op Translatorにこのリストを自動的に最新の状態に保ってもらいたい場合は、2つのコメントマーカーをそのまま残してください。
- 一部の言語のみ表示したい場合は、2つのコメントマーカーを削除し、表示したくない言語をリストから削除してください。マーカーを削除すると、Co‑op Translatorはこのセクションを自動で置き換えなくなります。
