# ಬೆಂಬಲ

## ಸಮಸ್ಯೆಗಳನ್ನು ದಾಖಲಿಸುವುದು ಮತ್ತು ಸಹಾಯವನ್ನು ಪಡೆಯುವುದು  

ಈ ಪ್ರಾಜೆಕ್ಟ್ ಬಗ್ಸ್ ಮತ್ತು ವೈಶಿಷ್ಟ್ಯಗಳ ವಿನಂತಿಗಳನ್ನು ಟ್ರ್ಯಾಕ್ ಮಾಡಲು GitHub Issues ಅನ್ನು ಬಳಸುತ್ತದೆ. ಡುಪ್ಲಿಕೇಟ್ಸ್‌ಗಳನ್ನು ತಪ್ಪಿಸಲು ಹೊಸ ಸಮಸ್ಯೆಗಳನ್ನು ದಾಖಲಿಸುವ ಮೊದಲು ಈಗಾಗಲೇ ಇರುವ ಸಮಸ್ಯೆಗಳನ್ನು ಹುಡುಕಿ. ಹೊಸ ಸಮಸ್ಯೆಗಳಿಗೆ, ನಿಮ್ಮ ಬಗ್ ಅಥವಾ ವೈಶಿಷ್ಟ್ಯ ವಿನಂತಿಯನ್ನು ಹೊಸ Issue ಆಗಿ ದಾಖಲಿಸಿ.

ಈ ಪ್ರಾಜೆಕ್ಟ್ ಅನ್ನು ಬಳಸುವ ಬಗ್ಗೆ ಸಹಾಯ ಮತ್ತು ಪ್ರಶ್ನೆಗಳಿಗೆ, ದಯವಿಟ್ಟು **ರೆಪೊ ನಿರ್ವಹಣಾಕಾರರು: ಸಹಾಯಕ್ಕಾಗಿ ರೆಪೊ ಮಾಲೀಕರು ಅಥವಾ ಸಮುದಾಯವನ್ನು ಸಂಪರ್ಕಿಸಲು ಸೂಚನೆಗಳನ್ನು ಇಲ್ಲಿ ಸೇರಿಸಿ. ಇದು Stack Overflow ಟ್ಯಾಗ್ ಅಥವಾ ಇತರ ಚಾನಲ್ ಆಗಿರಬಹುದು. ನೀವು ಜನರಿಗೆ ಸಹಾಯ ಮಾಡುವ ಸ್ಥಳ ಯಾವುದು?**.

## Microsoft ಬೆಂಬಲ ನೀತಿ  

ಈ **ಪ್ರಾಜೆಕ್ಟ್ ಅಥವಾ ಉತ್ಪನ್ನ**ಕ್ಕೆ ಬೆಂಬಲವು ಮೇಲಿನ ಸಂಪತ್ತಿಗೆ ಮಾತ್ರ ಸೀಮಿತವಾಗಿದೆ.
