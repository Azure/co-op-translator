# Co-op ಅನುವಾದಕ ಪ್ಯಾಕೇಜ್ ಅನ್ನು ಸ್ಥಾಪಿಸಿ

**Co-op Translator** ಒಂದು ಕಮಾಂಡ್-ಲೈನ್ ಇಂಟರ್ಫೇಸ್ (CLI) ಸಾಧನವಾಗಿದೆ, ಇದು ನಿಮ್ಮ ಪ್ರಾಜೆಕ್ಟ್‌ನಲ್ಲಿರುವ ಎಲ್ಲಾ markdown ಫೈಲ್‌ಗಳು ಮತ್ತು ಚಿತ್ರಗಳನ್ನು ಬಹುಭಾಷೆಗಳಲ್ಲಿ ಅನುವಾದಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಈ ಟ್ಯುಟೋರಿಯಲ್ ಅನುವಾದಕವನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡುವುದು ಮತ್ತು ವಿವಿಧ ಬಳಕೆ ಪ್ರಕರಣಗಳಿಗೆ ಅದನ್ನು ಚಲಾಯಿಸುವ ಪ್ರಕ್ರಿಯೆಯನ್ನು ನಿಮಗೆ ಮಾರ್ಗದರ್ಶನ ಮಾಡುತ್ತದೆ.

### ವರ್ಚುವಲ್ ಪರಿಸರವನ್ನು ರಚಿಸಿ

ನೀವು `pip` ಅಥವಾ `Poetry` ಬಳಸಿ ವರ್ಚುವಲ್ ಪರಿಸರವನ್ನು ರಚಿಸಬಹುದು. ನಿಮ್ಮ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಆಜ್ಞೆಗಳಲ್ಲಿ ಒಂದನ್ನು ಟೈಪ್ ಮಾಡಿ.

#### pip ಬಳಸಿ

```bash
python -m venv .venv
```

#### Poetry ಬಳಸಿ

```bash
poetry init
```

### ವರ್ಚುವಲ್ ಪರಿಸರವನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ

ವರ್ಚುವಲ್ ಪರಿಸರವನ್ನು ರಚಿಸಿದ ನಂತರ, ನೀವು ಅದನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಬೇಕಾಗುತ್ತದೆ. ನಿಮ್ಮ ಆಪರೇಟಿಂಗ್ ಸಿಸ್ಟಮ್ ಆಧರಿಸಿ ಹಂತಗಳು ವಿಭಿನ್ನವಾಗಿರುತ್ತವೆ. ನಿಮ್ಮ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಆಜ್ಞೆಯನ್ನು ಟೈಪ್ ಮಾಡಿ.

#### pip ಮತ್ತು Poetry ಎರಡರಿಗೂ

- ವಿಂಡೋಸ್:

    ```bash
    .venv\Scripts\activate
    ```

- ಮ್ಯಾಕ್/ಲಿನಕ್ಸ್:

    ```bash
    source .venv/bin/activate
    ```

#### Poetry ಬಳಸಿ

1. ನೀವು ಪರಿಸರವನ್ನು Poetry ಬಳಸಿ ರಚಿಸಿದ್ದರೆ, ಅದನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಲು ನಿಮ್ಮ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಆಜ್ಞೆಯನ್ನು ಟೈಪ್ ಮಾಡಿ.

    ```bash
    poetry shell
    ```

### ಪ್ಯಾಕೇಜ್ ಮತ್ತು ಅಗತ್ಯ ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸ್ಥಾಪಿಸುವುದು

ನಿಮ್ಮ ವರ್ಚುವಲ್ ಪರಿಸರವನ್ನು ರಚಿಸಿ ಮತ್ತು ಸಕ್ರಿಯಗೊಳಿಸಿದ ನಂತರ, ಮುಂದಿನ ಹಂತವು ಅಗತ್ಯ ಡಿಪೆಂಡೆನ್ಸಿಗಳನ್ನು ಸ್ಥಾಪಿಸುವುದು.

### ತ್ವರಿತ ಸ್ಥಾಪನೆ

Co-Op Translator ಅನ್ನು pip ಮೂಲಕ ಸ್ಥಾಪಿಸಿ

```
pip install co-op-translator
```
ಅಥವಾ 

Poetry ಮೂಲಕ ಸ್ಥಾಪಿಸಿ
```
poetry add co-op-translator
```

#### pip ಬಳಸಿ (requirements.txt ನಿಂದ) ನೀವು ಈ repo ಅನ್ನು ಕ್ಲೋನ್ ಮಾಡಿದರೆ 

> [!NOTE]
> ದಯವಿಟ್ಟು co-op translator ಅನ್ನು ತ್ವರಿತ ಸ್ಥಾಪನೆಯ ಮೂಲಕ ಸ್ಥಾಪಿಸಿದರೆ ಇದು ಮಾಡಬೇಡಿ.

1. ನೀವು pip ಬಳಸುತ್ತಿದ್ದರೆ, ನಿಮ್ಮ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಆಜ್ಞೆಯನ್ನು ಟೈಪ್ ಮಾಡಿ. ಇದು `requirements.txt` ಫೈಲ್‌ನಲ್ಲಿ ನಿರ್ದಿಷ್ಟಪಡಿಸಿದ ಅಗತ್ಯ ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸ್ಥಾಪಿಸುತ್ತದೆ:

    ```bash
    pip install -r requirements.txt
    ```

#### Poetry ಬಳಸಿ (pyproject.toml ನಿಂದ)

1. ನೀವು Poetry ಬಳಸುತ್ತಿದ್ದರೆ, ನಿಮ್ಮ ಟರ್ಮಿನಲ್‌ನಲ್ಲಿ ಈ ಕೆಳಗಿನ ಆಜ್ಞೆಯನ್ನು ಟೈಪ್ ಮಾಡಿ. ಇದು `pyproject.toml` ಫೈಲ್‌ನಲ್ಲಿ ನಿರ್ದಿಷ್ಟಪಡಿಸಿದ ಅಗತ್ಯ ಪ್ಯಾಕೇಜ್‌ಗಳನ್ನು ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಸ್ಥಾಪಿಸುತ್ತದೆ:

    ```bash
    poetry install
    ```
