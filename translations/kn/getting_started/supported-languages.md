# ಬೆಂಬಲಿತ ಭಾಷೆಗಳು

ಕೆಳಗಿನ ಟೇಬಲ್‌ನಲ್ಲಿ **Co-op Translator** ಪ್ರಸ್ತುತ ಬೆಂಬಲಿಸುವ ಭಾಷೆಗಳ ಪಟ್ಟಿ ನೀಡಲಾಗಿದೆ. ಇದರಲ್ಲಿ ಭಾಷಾ ಕೋಡ್‌ಗಳು, ಭಾಷಾ ಹೆಸರುಗಳು ಮತ್ತು ಪ್ರತಿ ಭಾಷೆಗೆ ಸಂಬಂಧಿಸಿದ ಯಾವುದೇ ಪರಿಚಿತ ಸಮಸ್ಯೆಗಳ ವಿವರಗಳನ್ನು ಒಳಗೊಂಡಿದೆ. ನೀವು ಹೊಸ ಭಾಷೆಯನ್ನು ಸೇರಿಸಲು ಇಚ್ಛಿಸುತ್ತಿದ್ದರೆ, `src/co_op_translator/fonts/` ನಲ್ಲಿ ಇರುವ `font_language_mappings.yml` ಫೈಲ್‌ನಲ್ಲಿ ಸಂಬಂಧಿತ ಭಾಷಾ ಕೋಡ್, ಹೆಸರು ಮತ್ತು ಸೂಕ್ತವಾದ ಫಾಂಟ್ ಸೇರಿಸಿ, ಪರೀಕ್ಷಿಸಿದ ನಂತರ ಪುಲ್ ರಿಕ್ವೆಸ್ಟ್ ಸಲ್ಲಿಸಿ.
