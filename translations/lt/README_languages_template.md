# 🌐 Daugiakalbė palaikymas (šablonas)

Prižiūrėtojams: žemiau esantis blokas yra „visų kalbų“ pavyzdys, kurį tvarko Co‑op Translator.

- Jei norite, kad Co‑op Translator automatiškai palaikytų šį sąrašą visiškai atnaujintą, kai paleidžiate `translate -l "all"`, palikite abu komentaro žymeklius nepakitusius.
- Jei norite rodyti tik tam tikras kalbas, ištrinkite abu komentaro žymeklius ir pašalinkite tas kalbas, kurių nenorite matyti sąraše. Pašalinus žymeklius, Co‑op Translator daugiau nebeatnaujins šios sekcijos automatiškai.
