# Palaikomos kalbos

Žemiau pateiktoje lentelėje išvardytos kalbos, kurias šiuo metu palaiko **Co-op Translator**. Joje nurodyti kalbų kodai, kalbų pavadinimai ir žinomos problemos, susijusios su kiekviena kalba. Jei norite pridėti naują kalbą, prašome įrašyti atitinkamą kalbos kodą, pavadinimą ir tinkamą šriftą faile `font_language_mappings.yml`, esančiame `src/co_op_translator/fonts/`, ir pateikti pull request po testavimo.
