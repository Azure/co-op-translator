# പിന്തുണ

## പ്രശ്നങ്ങൾ എങ്ങനെ ഫയൽ ചെയ്യാം, സഹായം എങ്ങനെ ലഭിക്കാം  

ഈ പ്രോജക്റ്റ് ബഗുകളും ഫീച്ചർ അഭ്യർത്ഥനകളും ട്രാക്ക് ചെയ്യാൻ GitHub Issues ഉപയോഗിക്കുന്നു. ഡുപ്ലിക്കേറ്റുകൾ ഒഴിവാക്കാൻ പുതിയ പ്രശ്നങ്ങൾ ഫയൽ ചെയ്യുന്നതിന് മുമ്പ് നിലവിലുള്ള പ്രശ്നങ്ങൾ തിരയുക. പുതിയ പ്രശ്നങ്ങൾക്ക്, നിങ്ങളുടെ ബഗ് അല്ലെങ്കിൽ ഫീച്ചർ അഭ്യർത്ഥന ഒരു പുതിയ Issue ആയി ഫയൽ ചെയ്യുക.

ഈ പ്രോജക്റ്റ് ഉപയോഗിക്കുന്നതിനെക്കുറിച്ചുള്ള സഹായത്തിനും ചോദ്യങ്ങൾക്കും, **റിപ്പോ പരിപാലകൻ: സഹായം ലഭിക്കാൻ റിപ്പോ ഉടമകളുമായി അല്ലെങ്കിൽ കമ്മ്യൂണിറ്റിയുമായി എങ്ങനെ ബന്ധപ്പെടാം എന്നതിന് നിർദ്ദേശങ്ങൾ ഇവിടെ ചേർക്കുക. ഇത് ഒരു Stack Overflow ടാഗ് അല്ലെങ്കിൽ മറ്റേതെങ്കിലും ചാനൽ ആകാം. നിങ്ങൾ എവിടെ ആളുകളെ സഹായിക്കും?**.

## Microsoft പിന്തുണ നയം  

ഈ **പ്രോജക്റ്റ് അല്ലെങ്കിൽ ഉൽപ്പന്നം** സംബന്ധിച്ച പിന്തുണ മുകളിൽ ലിസ്റ്റ് ചെയ്ത റിസോഴ്‌സുകൾക്ക് പരിമിതമാണ്.
