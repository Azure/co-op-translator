# Co-op Translator പാക്കേജ് ഇൻസ്റ്റാൾ ചെയ്യുക

**Co-op Translator** എന്നത് ഒരു കമാൻഡ്-ലൈൻ ഇന്റർഫേസ് (CLI) ഉപകരണമാണ്, നിങ്ങളുടെ പ്രോജക്റ്റിലെ എല്ലാ markdown ഫയലുകളും ചിത്രങ്ങളും പല ഭാഷകളിലേക്കും വിവർത്തനം ചെയ്യാൻ സഹായിക്കുന്നതിന് രൂപകൽപ്പന ചെയ്തിരിക്കുന്നത്. ഈ ട്യൂട്ടോറിയൽ വിവർത്തകനെ കോൺഫിഗർ ചെയ്യുന്നതും വിവിധ ഉപയോഗ കേസുകൾക്കായി അത് പ്രവർത്തിപ്പിക്കുന്നതും നിങ്ങളെ വഴി കാണിക്കും.

### ഒരു വെർച്വൽ എൻവയോൺമെന്റ് സൃഷ്ടിക്കുക

`pip` അല്ലെങ്കിൽ `Poetry` ഉപയോഗിച്ച് നിങ്ങൾക്ക് ഒരു വെർച്വൽ എൻവയോൺമെന്റ് സൃഷ്ടിക്കാം. നിങ്ങളുടെ ടെർമിനലിൽ താഴെ പറയുന്ന കമാൻഡുകളിൽ ഒന്നെഴുതുക.

#### pip ഉപയോഗിച്ച്

```bash
python -m venv .venv
```

#### Poetry ഉപയോഗിച്ച്

```bash
poetry init
```

### വെർച്വൽ എൻവയോൺമെന്റ് സജീവമാക്കുക

വെർച്വൽ എൻവയോൺമെന്റ് സൃഷ്ടിച്ച ശേഷം, നിങ്ങൾ അത് സജീവമാക്കേണ്ടതുണ്ട്. നിങ്ങളുടെ ഓപ്പറേറ്റിംഗ് സിസ്റ്റം അനുസരിച്ച് ചുവടെയുള്ള ചുവടുപടികൾ വ്യത്യാസപ്പെടുന്നു. നിങ്ങളുടെ ടെർമിനലിൽ താഴെ പറയുന്ന കമാൻഡ് ടൈപ്പ് ചെയ്യുക.

#### pip, Poetry രണ്ടിലും

- Windows:

    ```bash
    .venv\Scripts\activate
    ```

- Mac/Linux:

    ```bash
    source .venv/bin/activate
    ```

#### Poetry ഉപയോഗിച്ച്

1. നിങ്ങൾ Poetry ഉപയോഗിച്ച് എൻവയോൺമെന്റ് സൃഷ്ടിച്ചെങ്കിൽ, അത് സജീവമാക്കാൻ നിങ്ങളുടെ ടെർമിനലിൽ താഴെ പറയുന്ന കമാൻഡ് ടൈപ്പ് ചെയ്യുക.

    ```bash
    poetry shell
    ```

### പാക്കേജും ആവശ്യമായ പാക്കേജുകളും ഇൻസ്റ്റാൾ ചെയ്യുക

നിങ്ങളുടെ വെർച്വൽ എൻവയോൺമെന്റ് സജ്ജീകരിക്കുകയും സജീവമാക്കുകയും ചെയ്ത ശേഷം, അടുത്ത ഘട്ടം ആവശ്യമായ ഡിപൻഡൻസികൾ ഇൻസ്റ്റാൾ ചെയ്യുന്നതാണ്.

### പെട്ടെന്ന് ഇൻസ്റ്റാൾ ചെയ്യുക

Co-Op Translator പിപ് വഴി ഇൻസ്റ്റാൾ ചെയ്യുക

```
pip install co-op-translator
```
അല്ലെങ്കിൽ 

Poetry വഴി ഇൻസ്റ്റാൾ ചെയ്യുക
```
poetry add co-op-translator
```

#### ഈ റീപോ ക്ലോൺ ചെയ്താൽ pip (requirements.txt വഴി) ഉപയോഗിച്ച് 

> [!NOTE]
> ദയവായി, പെട്ടെന്ന് ഇൻസ്റ്റാൾ വഴി co-op translator ഇൻസ്റ്റാൾ ചെയ്താൽ ഇത് ചെയ്യരുത്.

1. നിങ്ങൾ pip ഉപയോഗിക്കുന്നുവെങ്കിൽ, നിങ്ങളുടെ ടെർമിനലിൽ താഴെ പറയുന്ന കമാൻഡ് ടൈപ്പ് ചെയ്യുക. ഇത് `requirements.txt` ഫയലിൽ വ്യക്തമാക്കിയ ആവശ്യമായ പാക്കേജുകൾ സ്വയമേവ ഇൻസ്റ്റാൾ ചെയ്യും:

    ```bash
    pip install -r requirements.txt
    ```

#### Poetry ഉപയോഗിച്ച് (pyproject.toml വഴി)

1. നിങ്ങൾ Poetry ഉപയോഗിക്കുന്നുവെങ്കിൽ, നിങ്ങളുടെ ടെർമിനലിൽ താഴെ പറയുന്ന കമാൻഡ് ടൈപ്പ് ചെയ്യുക. ഇത് `pyproject.toml` ഫയലിൽ വ്യക്തമാക്കിയ ആവശ്യമായ പാക്കേജുകൾ സ്വയമേവ ഇൻസ്റ്റാൾ ചെയ്യും:

    ```bash
    poetry install
    ```
