# समर्थित भाषा

खालील तक्ता सध्या **Co-op Translator** द्वारे समर्थित भाषांची यादी दर्शवितो. यात भाषा कोड, भाषा नावे आणि प्रत्येक भाषेशी संबंधित कोणत्याही ज्ञात समस्या समाविष्ट आहेत. जर तुम्हाला नवीन भाषा जोडायची असेल, तर कृपया `src/co_op_translator/fonts/` मध्ये असलेल्या `font_language_mappings.yml` फाईलमध्ये संबंधित भाषा कोड, नाव आणि योग्य फॉन्ट जोडा आणि चाचणी केल्यानंतर पुल रिक्वेस्ट सबमिट करा.
