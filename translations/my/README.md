## Жалпы шолу: Білім беру мазмұнын аударуды оңтайландыру

Тілдік кедергілер әлемдегі оқушылар мен әзірлеушілер үшін маңызды білім мен техникалық ресурстарға қол жеткізуді айтарлықтай шектейді. Бұл қатысуды төмендетіп, жаһандық инновация мен оқудың қарқынын бәсеңдетеді.

**Co-op Translator** Microsoft-тың өз ірі білім беру серияларының (мысалы, "Бастауыштарға арналған" нұсқаулықтар) қолмен аудару процесінің тиімсіздігін шешу қажеттігінен туындады. Қазіргі таңда бұл кедергілерді баршаға жоюға бағытталған, қолдануға жеңіл әрі қуатты құралға айналды. CLI мен GitHub Actions арқылы сапалы автоматты аудармаларды ұсына отырып, Co-op Translator оқытушыларға, студенттерге, зерттеушілерге және әзірлеушілерге тілдік шектеулерсіз білім алмасуға мүмкіндік береді.

Co-op Translator аударылған білім беру мазмұнын қалай ұйымдастыратынын қараңыз:

![Example](../../translated_images/translation-ex.e97fcdf09d27f56060939080a0b3e491e92b67e047ca005826cb3de11399907c.my.png)

Markdown файлдары мен суреттердегі мәтін автоматты түрде аударылып, тілге арнайы папкаларға реттеледі.

**Co-op Translator арқылы білім беру мазмұныңызға жаһандық қолжетімділікті бүгін ашыңыз!**

## Негізгі ерекшеліктер

- **Автоматты аудармалар**: Мәтінді бірнеше тілге оңай аударады.
- **GitHub Actions интеграциясы**: CI/CD процесіңіздің бір бөлігі ретінде аудармаларды автоматтандырады.
- **Markdown сақталуы**: Аударма кезінде Markdown синтаксисі бұзылмайды.
- **Суреттегі мәтінді аудару**: Суреттердегі мәтінді шығарып, аударады.
- **Жетілдірілген LLM технологиясы**: Сапалы аудармалар үшін соңғы тіл модельдерін пайдаланады.
- **Оңай интеграция**: Қолданыстағы жобаңызға оңай қосылады.
- **Локализацияны жеңілдету**: Жобаңызды халықаралық нарыққа бейімдеуді оңтайландырады.

## Қалай жұмыс істейді

![Architecture](../../translated_images/architecture_241019.15de09d63bf3f7065d5da8ac1e08948c9d3bf795aae3b7f6f8d3cce53c95d00b.my.png)

Co-op Translator сіздің жобаңыздағы Markdown файлдары мен суреттерді келесі қадамдармен өңдейді:

1. **Мәтінді шығару**: Markdown файлдарынан және (мысалы, Azure AI Vision арқылы) суреттердегі мәтінді шығарады.
1. **AI аударма**: Шығарылған мәтінді конфигурацияланған LLM-ге (Azure OpenAI, OpenAI және т.б.) аударады.
1. **Нәтижені сақтау**: Аударылған Markdown файлдары мен суреттерді тілге сәйкес папкаларға сақтайды, бастапқы форматты сақтай отырып.

## Бастау

CLI арқылы тез бастауға немесе GitHub Actions көмегімен толық автоматтандыруды орнатуға болады. Өз жұмыс үдерісіңізге ең қолайлы әдісті таңдаңыз:

Бастамас бұрын, келесі ресурстарды орнату қажет:

1. Тіл моделі ресурсы (міндетті):

### Жылдам бастау: Командалық жол

Командалық жол арқылы тез бастау үшін:

1. Виртуалды орта жасаңыз:

    ```bash
    python -m venv .venv
    ```

1. Виртуалды ортаны іске қосыңыз:

    - Windows жүйесінде:

    ```bash
    .venv\scripts\activate
    ```

    - Linux/macOS жүйесінде:

    ```bash
    source .venv/bin/activate
    ```

1. Пакетті орнатыңыз:

    ```bash
    pip install co-op-translator
    ```

1. Құпия деректерді баптаңыз:

    - `.env` file in your project's root directory.
    - Copy the contents from the [.env.template](../../.env.template) file into your new `.env` file.
    - Fill in the required API keys and endpoint information in your `.env` file.

1. Run Translation:
    - Navigate to your project's root directory in your terminal.
    - Execute the translate command, specifying target languages with the `-l` туынын жасаңыз:

    ```bash
    translate -l "ko ja fr"
    ```

_(“ko ja fr” дегенді өз қажетті тілдеріңізге ауыстырыңыз)_

## Responsible AI

Microsoft is dedicated to helping customers use our AI products responsibly, sharing what we’ve learned, and building trust through tools like Transparency Notes and Impact Assessments. Many resources are available at [https://aka.ms/RAI](https://aka.ms/RAI).  
Our responsible AI approach is based on principles of fairness, reliability and safety, privacy and security, inclusiveness, transparency, and accountability.

Large-scale natural language, image, and speech models—like those used in this example—can sometimes behave in ways that are unfair, unreliable, or offensive, potentially causing harm. Please review the [Azure OpenAI service Transparency note](https://learn.microsoft.com/legal/cognitive-services/openai/transparency-note?tabs=text) to understand the risks and limitations.

The best way to reduce these risks is to include a safety system in your design that can detect and block harmful behavior. [Azure AI Content Safety](https://learn.microsoft.com/azure/ai-services/content-safety/overview) offers an independent protection layer that detects harmful user-generated and AI-generated content in apps and services. It includes text and image APIs for identifying harmful material. There’s also an interactive Content Safety Studio where you can explore and test sample code for detecting harmful content across various formats. The following [quickstart guide](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-text?tabs=visual-studio%2Clinux&pivots=programming-language-rest) will walk you through making requests to the service.

Another important factor is overall application performance. For multi-modal and multi-model applications, performance means the system behaves as expected, including avoiding harmful outputs. It’s crucial to evaluate your application’s performance using [generation quality and risk and safety metrics](https://learn.microsoft.com/azure/ai-studio/concepts/evaluation-metrics-built-in).

You can assess your AI application in your development environment using the [prompt flow SDK](https://microsoft.github.io/promptflow/index.html). Whether you have a test dataset or a target, your generative AI outputs can be quantitatively evaluated using built-in or custom evaluators. To get started with the prompt flow SDK for evaluation, follow the [quickstart guide](https://learn.microsoft.com/azure/ai-studio/how-to/develop/flow-evaluate-sdk). After running an evaluation, you can [visualize the results in Azure AI Studio](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-flow-results).
