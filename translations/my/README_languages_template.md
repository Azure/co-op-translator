# 🌐 ဘာသာစကားများစွာ ထောက်ပံ့မှု (နမူနာ)

ထိန်းသိမ်းသူများအတွက် - အောက်ပါ block သည် Co‑op Translator မှ စီမံခန့်ခွဲထားသော "ဘာသာစကားအားလုံး" နမူနာဖြစ်သည်။

- Co‑op Translator ကို အသုံးပြု၍ `translate -l "all"` ကို run လုပ်သည့်အခါ ဘာသာစကားစာရင်းကို အလိုအလျောက် update လုပ်စေချင်လျှင်၊ comment marker နှစ်ခုကို မပြောင်းပါနှင့်။
- ဘာသာစကားအချို့သာ ပြသချင်လျှင်၊ comment marker နှစ်ခုကို ဖျက်ပြီး မလိုအပ်သော ဘာသာစကားများကို ဖယ်ရှားပါ။ marker များကို ဖယ်ရှားပြီးနောက် Co‑op Translator သည် ဤအပိုင်းကို အလိုအလျောက် အစားထိုးမလုပ်တော့ပါ။
