## Een nieuwe taal toevoegen

Om ondersteuning toe te voegen voor een nieuwe taal:

1. Ga naar [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml).
2. Voeg de taalcode, naam en het juiste lettertypebestand toe. Zorg ervoor dat je het `rtl` attribuut toevoegt als de taal van rechts naar links wordt geschreven.
3. Als je een nieuw lettertype wilt gebruiken, controleer dan of het lettertype vrij te gebruiken is in open source projecten door de licentie- en copyrightvoorwaarden te controleren. Voeg na bevestiging het lettertypebestand toe aan de `src/co_op_translator/fonts/` map.
4. Test je wijzigingen lokaal om te controleren of de nieuwe taal correct wordt ondersteund.
5. Dien een Pull Request in met je wijzigingen en vermeld de toevoeging van de nieuwe taal in de beschrijving van de PR.

Voorbeeld:

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
