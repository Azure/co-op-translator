# 🌐 Suport pentru mai multe limbi (Șablon)

Responsabili: Blocul de mai jos este un exemplu „pentru toate limbile” gestionat de Co‑op Translator.

- Dacă vrei ca Co‑op Translator să mențină această listă complet actualizată automat când rulezi `translate -l "all"`, păstrează cele două marcaje de comentarii așa cum sunt.
- Dacă vrei să afișezi doar un subset de limbi, șterge cele două marcaje de comentarii și elimină limbile pe care nu vrei să le listezi. După ce elimini marcajele, Co‑op Translator nu va mai înlocui automat această secțiune.
