## Добавление нового языка

Чтобы добавить поддержку нового языка:

1. Перейдите в [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml).
2. Добавьте код языка, название и имя файла шрифта. Обязательно укажите атрибут `rtl`, если язык пишется справа налево.
3. Если необходимо использовать новый шрифт, убедитесь, что его лицензия позволяет бесплатное использование в проектах с открытым исходным кодом. После проверки добавьте файл шрифта в каталог `src/co_op_translator/fonts/`.
4. Проверьте свои изменения локально, чтобы убедиться, что новый язык корректно поддерживается.
5. Отправьте Pull Request с вашими изменениями и укажите добавление нового языка в описании PR.

Пример:

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
