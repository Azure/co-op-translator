# Sasisha sehemu ya "Kozi Nyingine" (Microsoft Beginners repos)

Mwongozo huu unaelezea jinsi ya kufanya sehemu ya "Kozi Nyingine" iendane moja kwa moja kwa kutumia Co-op Translator, na jinsi ya kusasisha kiolezo cha jumla kwa repos zote.

- Inahusu: Hifadhi za Microsoft Beginners pekee
- Inafanya kazi na: Co-op Translator CLI na GitHub Actions
- Chanzo cha kiolezo: [src/co_op_translator/templates/other_courses.md](../src/co_op_translator/templates/other_courses.md)

---

## Anza haraka: Washa ulandanishi wa moja kwa moja kwenye repo yako

Ongeza alama zifuatazo kuzunguka sehemu ya "Kozi Nyingine" kwenye README yako. Co-op Translator itabadilisha kila kitu kilicho kati ya alama hizi kila mara inapoendeshwa.


Kila wakati Co-op Translator inapoendeshwa—kupitia CLI (mfano, `translate -l "<language codes>"`) au GitHub Actions—inasasisha sehemu ya "Kozi Nyingine" iliyozungukwa na alama hizi moja kwa moja.

> [!NOTE]
> Ikiwa tayari una orodha iliyopo, ingiza tu ndani ya alama hizi. Mara inayofuata itabadilisha na kuweka maudhui ya kisasa yaliyo sanifu.

---

## Jinsi ya kubadilisha maudhui ya jumla

Ikiwa unataka kusasisha maudhui yaliyo sanifu yanayoonekana katika repos zote za Beginners:

1. Hariri kiolezo: [src/co_op_translator/templates/other_courses.md](../src/co_op_translator/templates/other_courses.md)
2. Fungua ombi la pull kwenye repo ya Co-op Translator na mabadiliko yako
3. Baada ya PR kuunganishwa, toleo la Co-op Translator litasasishwa
4. Mara inayofuata Co-op Translator itakapotekelezwa (CLI au GitHub Action) kwenye repo lengwa, italeta ulandanishi wa sehemu iliyosasishwa moja kwa moja

Hii inahakikisha chanzo kimoja cha ukweli kwa maudhui ya "Kozi Nyingine" katika hifadhi zote za Beginners.
