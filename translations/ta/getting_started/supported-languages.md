# ஆதரவு மொழிகள்

கீழே உள்ள அட்டவணையில் **Co-op Translator** தற்போது ஆதரவு வழங்கும் மொழிகள் பட்டியலிடப்பட்டுள்ளன. இதில் மொழி குறியீடுகள், மொழி பெயர்கள் மற்றும் ஒவ்வொரு மொழியுடனும் தொடர்புடைய அறியப்பட்ட பிரச்சனைகள் உள்ளன. புதிய மொழிக்கு ஆதரவு சேர்க்க விரும்பினால், `src/co_op_translator/fonts/` இல் உள்ள `font_language_mappings.yml` கோப்பில் பொருத்தமான மொழி குறியீடு, பெயர் மற்றும் சரியான எழுத்துருவை சேர்க்கவும், பின்னர் சோதனை செய்து புல் கோரிக்கையை சமர்ப்பிக்கவும்.
