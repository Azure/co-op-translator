# "மற்ற பாடங்கள்" பகுதியை புதுப்பிக்கவும் (Microsoft Beginners ரெப்போக்கள்)

இந்த வழிகாட்டி "மற்ற பாடங்கள்" பகுதியை Co-op Translator மூலம் தானாக ஒத்திசைக்க எப்படி செய்வது மற்றும் அனைத்து ரெப்போக்களுக்கும் உலகளாவிய வார்ப்புருவை எப்படி புதுப்பிப்பது என்பதை விளக்குகிறது.

- பொருந்தும்: Microsoft Beginners ரெப்போக்களுக்கு மட்டும்
- வேலை செய்கிறது: Co-op Translator CLI மற்றும் GitHub Actions உடன்
- வார்ப்புரு மூலம்: [src/co_op_translator/templates/other_courses.md](../src/co_op_translator/templates/other_courses.md)

---

## விரைவான தொடக்கம்: உங்கள் ரெப்போவில் தானாக ஒத்திசைவை இயக்கவும்

உங்கள் README இல் "மற்ற பாடங்கள்" பகுதியை கீழ்க்காணும் குறியீடுகளால் சுற்றி கொள்ளவும். Co-op Translator ஒவ்வொரு முறையும் இந்த குறியீடுகளுக்குள் உள்ள அனைத்தையும் மாற்றும்.

```markdown
<!-- CO-OP TRANSLATOR OTHER COURSES START -->
<!-- The content between START and END is auto-generated. Do not edit manually. -->
<!-- CO-OP TRANSLATOR OTHER COURSES END -->
```

Co-op Translator ஒவ்வொரு முறையும் இயக்கும் போது—CLI மூலம் (எ.கா., `translate -l "<language codes>"`) அல்லது GitHub Actions மூலம்—இந்த குறியீடுகளால் சுற்றப்பட்ட "மற்ற பாடங்கள்" பகுதியை தானாக புதுப்பிக்கும்.

> [!NOTE]
> ஏற்கனவே ஒரு பட்டியல் இருந்தால், அதனை அதே குறியீடுகளால் சுற்றி கொள்ளுங்கள். அடுத்த இயக்கம் அதனை சமீபத்திய ஒரே மாதிரியான உள்ளடக்கத்துடன் மாற்றும்.

## உலகளாவிய உள்ளடக்கத்தை எப்படி மாற்றுவது

அனைத்து Beginners ரெப்போக்களிலும் தோன்றும் ஒரே மாதிரியான உள்ளடக்கத்தை புதுப்பிக்க விரும்பினால்:

1. வார்ப்புருவை திருத்தவும்: [src/co_op_translator/templates/other_courses.md](../src/co_op_translator/templates/other_courses.md)
2. உங்கள் மாற்றங்களுடன் Co-op Translator ரெப்போவில் ஒரு pull request திறக்கவும்
3. PR இணைக்கப்பட்ட பிறகு, Co-op Translator பதிப்பு புதுப்பிக்கப்படும்
4. அடுத்த முறையும் Co-op Translator (CLI அல்லது GitHub Action) இலக்கு ரெப்போவில் இயக்கும் போது, அது தானாக புதுப்பிக்கப்பட்ட பகுதியை ஒத்திசைக்கும்

இது அனைத்து Beginners ரெப்போக்களிலும் "மற்ற பாடங்கள்" உள்ளடக்கத்திற்கு ஒரே உண்மையான மூலத்தை உறுதி செய்கிறது.
