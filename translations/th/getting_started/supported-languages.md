# ภาษาที่รองรับ

ตารางด้านล่างแสดงภาษาที่รองรับโดย **Co-op Translator** ในปัจจุบัน ซึ่งรวมถึงรหัสภาษา ชื่อภาษา และปัญหาที่ทราบเกี่ยวกับแต่ละภาษา หากคุณต้องการเพิ่มการรองรับภาษาใหม่ กรุณาเพิ่มรหัสภาษา ชื่อภาษา และฟอนต์ที่เหมาะสมในไฟล์ `font_language_mappings.yml` ที่อยู่ในโฟลเดอร์ `src/co_op_translator/fonts/` และส่งคำขอดึง (pull request) หลังจากทดสอบแล้ว
