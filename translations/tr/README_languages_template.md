# 🌐 Çok Dilli Destek (Şablon)

Bakımcılar: Aşağıdaki blok, Co‑op Translator tarafından yönetilen "tüm diller" örneğidir.

- Co‑op Translator'ın bu listeyi `translate -l "all"` komutunu çalıştırdığınızda tamamen güncel tutmasını istiyorsanız, iki yorum işaretini olduğu gibi bırakın.
- Sadece belirli dilleri göstermek istiyorsanız, iki yorum işaretini silin ve listelemek istemediğiniz dilleri kaldırın. İşaretleri kaldırdıktan sonra, Co‑op Translator bu bölümü artık otomatik olarak güncellemeyecek.
