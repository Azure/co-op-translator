## Adding a new language

新增語言支援步驟：

1. 前往 [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml)。
2. 新增語言代碼、名稱及對應的字體檔案名稱。如果語言是由右至左，請記得加入 `rtl` 屬性。
3. 若需使用新字體，請確認該字體在開源專案中可免費使用，並檢查授權及版權條款。確認無誤後，將字體檔案加入 `src/co_op_translator/fonts/` 目錄。
4. 在本地測試變更，確保新語言能正常支援。
5. 提交 Pull Request，並在 PR 描述中說明新增語言的內容。

Example:

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
