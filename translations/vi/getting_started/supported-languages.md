# Ngôn ngữ được hỗ trợ

Bảng dưới đây liệt kê các ngôn ngữ hiện đang được **Co-op Translator** hỗ trợ. Nó bao gồm mã ngôn ngữ, tên ngôn ngữ và các vấn đề đã biết liên quan đến từng ngôn ngữ. Nếu bạn muốn thêm hỗ trợ cho một ngôn ngữ mới, vui lòng thêm mã ngôn ngữ tương ứng, tên và font phù hợp vào tệp `font_language_mappings.yml` nằm trong thư mục `src/co_op_translator/fonts/` và gửi pull request sau khi đã thử nghiệm.
