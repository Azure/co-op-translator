## 添加新语言

要添加对新语言的支持：

1. 访问 [src/co_op_translator/fonts/font_language_mappings.yml](https://github.com/Azure/co-op-translator/blob/main/src/co_op_translator/fonts/font_language_mappings.yml)。
2. 添加语言代码、名称和合适的字体文件名。如果该语言是从右向左书写，请确保包含`rtl`属性。
3. 如果需要使用新字体，请确保该字体在开源项目中免费使用，检查其许可和版权条款。确认无误后，将字体文件添加到`src/co_op_translator/fonts/`目录。
4. 在本地测试更改，确保新语言得到正确支持。
5. 提交包含更改的拉取请求，并在PR描述中说明新增语言。

示例：

```yaml
new_lang:
  name: "New Language"
  font: "NotoSans-Medium.ttf"
  rtl: false
```
